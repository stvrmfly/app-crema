Motion principles

Hover

Color & shadow only — never transform. Cards lift via shadow change; icon buttons brighten via color. Neighbors never reflow on hover.

Duration scale

150–180 ms for color/bg, 200–300 ms for layout (cart collapse, sidebar width), 400–520 ms for shell-level entrance/exit animations.

Reduced motion

prefers-reduced-motion: reduce suppresses every custom animation utility we ship (entrance cascades, skeleton shimmer, tour pulse, spinner). The UI goes still without breaking layout.

Touch targets

Every icon-only button has a ≥44 × 44 px hit area via the .hit-target utility (invisible pseudo-element). Visual size is free to be smaller for table density.
